== Description ==
Provides conditionals for Polylang to check against the current active language

== Installation ==
Go to Plugins > Add New > Upload  and select the zip file of the WordPress plugin you wish to add.

== Frequently Asked Questions ==
Q: How does this plugin work?
A: After activation you can make use of the is_language( $lang ) function and/or the [is_language lang="en"]Content[/is_language] shortcode to display contents only for a specific language.
